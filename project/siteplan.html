Site Purpose

FitDash is a simple and intuitive fitness tracker dashboard designed to help users monitor their physical activity, track goals, and view progress in one convenient place.

Scenarios

Scenario 1: How can I log my daily workout?

The user can enter their workout details (e.g., type, duration, intensity) using the dashboard form.

Scenario 2: Can I see how many calories I burned this week?

Users can view a summary or graph of weekly calories burned.

Scenario 3: What’s my fitness goal progress?

The dashboard displays progress bars or percentages for set goals like steps walked, weight lost, or workouts completed.

Color Schema

Accent Colors:

  • #1E1E1E (Dark Charcoal) – For headings and contrast
  • #FFC300 (Bright Yellow) – For call-to-action buttons like “Book Now”
  • #D1E8E2 (Soft Teal) – For highlights, hover states, or notifications

Background Color: #F7FAFC (Light Gray)
